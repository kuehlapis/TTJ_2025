
# TTJ_2025

## Repository Setup

1. Clone the repository:
	```bash
	git clone <repo-url>
	cd TTJ_2025/frontend
	```

## Setup Instructions

2. Install dependencies:
	```bash
	npm i
	```

3. Run the frontend:
	```bash
	npm run dev
	```
