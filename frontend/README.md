# TTJ_2025
## Setup Instructions

1. Install dependencies:
	```bash
	npm i
	```

2. Run the frontend (from the `frontend` folder):
	```bash
	npm run dev
	```

After starting the development server, you can access the frontend at:

	➜  Local:   http://localhost:8080/
	➜  Network: http://192.168.10.137:8080/
